Linux安装OpenJDK

由于CentOS已停止更新优先建议使用Debian 11及以上操作系统，如需使用CentOS请采用CentOS 7.9，本次安装采用Debian系统。

# 1. JDK下载

OpenJDK下载：官方地址 (opens new window)

选择Linux对应版本进行下载 正常情况下载的文件名类似OpenJDK8U-jdk_x64_linux_hotspot_\*.tar.gz

# 2. 安装准备

# 2.1 更新操作系统

apt update && apt install vim -y && apt install zip -y && apt install unzip -y
1

# 2.2 创建目录

mkdir -p /data
1

# 2.3 上传文件

将下载好的JDK，上传文件至/data目录下

# 2.4 解压文件

# 解压JDK并重命名
cd /data/ && mkdir -p /data/jdk && tar -xzvf OpenJDK*.tar.gz -C /data/jdk --strip-components 1 && rm -rf OpenJDK*.tar.gz
1
2

# 3. 安装JDK

# 3.1 配置JDK环境变量

echo "export JAVA_HOME=/data/jdk
export CLASSPATH=.:\$JAVA_HOME/jre/lib/rt.jar:\$JAVA_HOME/lib/dt.jar:\$JAVA_HOME/lib/tools.jar
export PATH=\$PATH:\$JAVA_HOME/bin" >> /etc/profile
1
2
3

# 3.2 更新配置文件并验证版本

source /etc/profile && java -version
1
