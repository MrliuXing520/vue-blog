# 关于本站

本站名为荔枝笔记，是一个专注于记录技术文章和个人日常生活的博客网站。关于名字的选择一直以来都为此感到为难，觉得无论取什么名字都不够好听。突然间灵感迸发，想到了我第一次创业开发的第一个实战千人项目就叫荔枝，于是索性将博客命名为荔枝笔记。

# 关于博主

一个刚毕业初入社会的Java开发后端菜鸟，开发过一些项目，正在努力成长中。平时喜欢研究一些新的技术。

# 博客历程

  • 2016年6月

初中时期初接触博客，当时照着网上的各种教程搭建了第一个博客，记录生活中的点滴。

  • 2020年8月

走走停停，当时都是用免费空间免费域名，博客中途关闭了好多次，又换了好多次风格，主打一个折腾。

  • 2021年2月

博客换成了WordPress搭配各种主题，也开始记录写一些简单技术文章，博客总浏览数也破万了。

  • 2022年1月

始终感觉动态博客太臃肿，决定换成Hexo，之前的各种数据索性一律清空，重现开始。

  • 2024年1月

中途也是断断续续的更新，总觉得文章没有价值。又一次清空，摒弃了Hexo换成了VuePress，目前感觉很好，很简洁，估计很长一段时间不会换了，认真记录点滴了。
