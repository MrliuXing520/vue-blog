通过Vue-CLI创建Vue工程

创建一个 Vue 项目通常使用 Vue CLI（命令行界面）。Vue CLI 提供了一些命令和工具，帮助你创建、管理和构建 Vue 项目。以下是使用 Vue CLI 创建 Vue 项目的基本步。

# 1. 安装 Vue CLI

打开终端（命令行界面）并执行以下命令安装 Vue CLI。请确保你已经安装了 Node.js。

npm install -g @vue/cli
1

# 2. 创建新的 Vue 项目

使用以下命令创建一个新的 Vue 项目。在命令中，my-vue-project 是你想要创建的项目名称，你可以根据实际情况进行更改。

vue create my-vue-project
1

执行上述命令后，Vue CLI 会提示你选择一些配置，比如是否使用预设配置、选择特定的特性等。你可以选择默认配置，也可以根据项目需求进行配置。

# 3. 进入项目目录

完成项目创建后，使用以下命令进入项目目录：

cd my-vue-project
1

# 4. 启动开发服务器

在项目目录中，使用以下命令启动开发服务器：

npm run serve
1

这会启动一个本地开发服务器，并且你可以在浏览器中访问 http://localhost:8080 查看你的 Vue 应用。

这样就成功创建并启动了一个基本的 Vue 项目。可以根据实际需求进行进一步的开发和配置。有关 Vue CLI 更多的功能和选项，你可以查阅 Vue CLI 的官方文档：Vue CLI 官方文档 (opens new window)。
